PLUME/RUSE

Plume/Ruse are two compositions created using a combination of modular systems and other digital and hardware tools. The first, Plume, was composed using oscillator, filter, delay, and mixer modules, controlled by an Arturia MicroBrute. Both interations of the second piece, Ruse, were created with a combination of hardware and software. The primary drone is created on a Make Noise tELHARMONIC module, run through a Blue Sky hardware delay. Other oscillators are given pitch and parameter inputs generated in Max/MSP in a genetic algorithm process. These MIDI data are converted to C.V with an oplab and sent to an oscillator module. The drum programming has rhythms generated in a Euclidean sequencer, routed through a software drum machine, before being output through a Cathedral hardware reverb unit.
